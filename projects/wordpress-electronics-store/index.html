Lighting & Electronics Store#

Overview#

This project is a modern, fully responsive WordPress e-commerce website developed for selling a wide range of lighting products, electrical devices, and electronic equipment.
The platform emphasizes a clean and intuitive interface, fast performance, and a seamless shopping experience, enabling customers to easily browse, compare, and purchase products.

Key Features#

  • Full WooCommerce shop setup for lighting and electronics
  • Product categories & variations (wattage, color, type, size)
  • Advanced search & filtering for quick product discovery
  • Smooth checkout experience with multiple payment options
  • Customer accounts with order history and profile management
  • Responsive design optimized for mobile, tablet, and desktop
  • Homepage highlighting featured and best-selling products

Technologies Used#

  • WordPress (latest version)
  • WooCommerce
  • PHP 8+ / MySQL
  • HTML5, CSS3, JavaScript
  • Custom-designed responsive theme
  • Key plugins: SEO, Cache, Contact Form 7, Zarinpal (Iranian Payment), Security plugin, etc.
